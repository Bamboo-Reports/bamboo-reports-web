# Progress Handoff — 2 July 2026

## Current state

The public GCC Tracker is available at `/tracker`. It sizes an India GCC market using aggregate counts for accounts, centres, and decision-makers.

### Filters

1. **Industry**
   - Database field: `accounts.account_primary_category`
   - Multi-select

2. **City**
   - Database field: `centers.center_city`
   - Multi-select

3. **Company**
   - Database field: `accounts.account_global_legal_name`
   - Single-company autocomplete
   - Starts searching after two characters
   - Returns up to eight suggestions
   - Prefix matches rank before substring matches
   - Industry and City selections constrain suggestions

Filters use OR within a multi-select and AND across filters. Selecting a company filters all three result counts.

## Tracker architecture

```text
Browser
  POST /api/tracker
    {
      account_primary_category: string[],
      center_city: string[],
      account_global_legal_name: string[],
      account_search: string
    }
        |
        v
Netlify function: netlify/functions/tracker.js
        |
        v
Neon PostgreSQL: accounts + centers + prospects
        |
        v
Counts and cascading facets, including limited company suggestions
```

The company autocomplete is debounced by 250 ms. It does not send the full company directory to the browser.
